About Name Sort

Name Sort is a option that makes alike named files be put together.

The threshold is how alike the names have to be for them to be put together.

The number represents the amount of numbers needed for files to be considered alike enough to be sorted together.

For example, a threshold of 5 will make 'image.png' and 'image2.png' be put together in an 'image' folder, but imag.png will not.
