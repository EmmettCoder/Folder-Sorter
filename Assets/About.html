About Folder-Sorter

Folder-Sorter is a lightweight utility designed to automatically organize cluttered directories.

Whether your Downloads folder is a mess or your project files are scattered, this tool provides a simple, customizable way to bring order to your files.

Features

  • Flexible Sorting: You can manually select any directory on your computer for the program to clean.
  • One Click Cleaning: The program can instantly sort the folder it is currently located in, making it perfect for keeping your Downloads folder tidy.
  • Custom Rules: You define exactly where files go based on their extensions using the configuration file.
  • Nested Organization: Supports creating subdirectories within folders (e.g., moving .png files into Images/pngs) to ensure a deep level of organization.
